# StartShield ADHD App

A simple, local ADHD focus tool built with plain HTML, CSS, and JavaScript.

## Features
- **Focus Timer**: A 25-minute Pomodoro-style timer to help manage focus sessions.
- **Task Setting**: Set a single clear goal to maintain focus during your session.

## Usage
Open `index.html` in any modern web browser to start using the app. No server or installation required.
